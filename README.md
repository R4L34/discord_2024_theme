# Discord 2024 Theme
Brings back the the discord before the theme update in 2025 while keeping it modern (unlike the original oldcord).

# How to use
Download vencord or any other external tool that allows you to load custom css into the Discord app. 
And put the css files from the repo where needed.
I'm not gonna explain how to use vencord, there are plenty youtube tutorials for that.

I myself use those themes, so if any update breaks something, I'll most likely fix it quick enough.
